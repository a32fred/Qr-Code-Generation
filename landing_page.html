API Documentation

Quick Start

# Get your API key
curl -X POST https://qrapi.dev/api/register

# Generate a QR code
curl -X POST https://qrapi.dev/api/generate \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "data": "https://example.com",
    "size": 512,
    "color": "#3B82F6",
    "format": "png"
  }'

Parameters

data

The text or URL to encode (required)

size

Size in pixels (default: 256)

color

Foreground color (hex, Pro+)

bg_color

Background color (hex, Pro+)

logo

Base64 encoded logo (Pro+)

Response

{
  "qr_code": "iVBORw0KGgoAAAANSUhEUgAA...", // base64 encoded image
  "qr_url": "https://qrapi.dev/qr/abc123",    // Direct URL
  "analytics": "https://qrapi.dev/analytics/abc123" // Analytics URL
}
